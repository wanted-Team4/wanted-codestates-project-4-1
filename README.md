## 🧥 Code Convention

### Getting Started

1. `clone` the repository,

```
$ git clone "https://github.com/wanted-Team4/wanted-codestates-project-4-1.git"
```

2. `Install` dependencies,

```
$ npm install
```

3. `Setting` prettier,

```
$ npx prettier --write .
```

4. `start` the project,

```
$ npm start
```
